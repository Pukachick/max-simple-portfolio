# Worapon Max - Data Analyst Portfolio

A modern, minimalist portfolio website designed for data analysts. Built with plain HTML and CSS, featuring a JetBrains Mono aesthetic and a built-in dark/light theme toggle.

## 🌟 Features
- **Modern Dark Theme**: Default professional dark aesthetic.
- **Theme Toggle**: Switch between dark and light modes with a single click (Sun/Moon icons).
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
- **Data Focused**: Clean typography using JetBrains Mono for a technical feel.
- **Performance**: Lightweight, vanilla implementation with zero dependencies.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Grid, and Flexbox.
- **JavaScript**: Minimalist implementation for theme switching.
- **Google Fonts**: JetBrains Mono.

## 🚀 Getting Started
To view the project locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/Pukachick/max-simple-portfolio.git
   ```
2. Open `index.html` in your favorite web browser.

## 🌐 Deployment
This project is ready for **GitHub Pages**. To deploy:
1. Go to your repository settings on GitHub.
2. Select **Pages** from the sidebar.
3. Choose the `main` branch and the `/ (root)` folder.
4. Click **Save**. Your site will be live in a few minutes!

## 📂 Project Structure
- `index.html`: Main site content and structure.
- `style.css`: All styling, theme variables, and responsive rules.
- `PLAN.md`: Detailed project design and implementation roadmap.
