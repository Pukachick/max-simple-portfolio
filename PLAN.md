# Project Plan: Data Analyst Portfolio Website

## 1. Vision & Goals
Build a professional, high-performance portfolio website for **Worapon Max**, a Data Analyst. The site is designed to showcase technical skills (Python, SQL, Data Visualization) and high-impact projects while maintaining a modern, minimalist aesthetic.

## 2. Technical Stack
- **Core**: Vanilla HTML5, CSS3, JavaScript (ES6+).
- **Typography**: JetBrains Mono (via Google Fonts) for a "developer/data" aesthetic.
- **Hosting**: Optimized for GitHub Pages.
- **Version Control**: Git.

## 3. Key Features
- **Modern Dark Theme**: Default high-contrast dark mode for reduced eye strain and professional look.
- **Theme Toggle**: Circular SVG-based toggle for switching between Light and Dark modes.
- **Responsive Design**: Mobile-first layout using CSS Flexbox and Grid.
- **Project Showcase**: Grid-based layout for 3+ featured projects with tagging and direct links.
- **Contact Integration**: Minimalist footer with Email, GitHub, and LinkedIn links.

- [x] Initialized Git repository.

- [x] Implemented JetBrains Mono typography.

## 5. Deployment Guide
1. Push the code to a new GitHub repository.
2. Enable GitHub Pages in the repository settings (Source: `main` branch, `/root` folder).
3. Update project descriptions in `index.html` as new work is completed.
